Capabilities

Everything a diagnostic platform should be.

Built for real shops. Powered by local AI. No internet required for core diagnostics.

Context Window Orchestration

The CWOP engine manages a structured context window with dedicated slots for DTCs, sensor data, vehicle info, and analysis — giving the LLM exactly what it needs, nothing it doesn't.

Core Architecture

Real-Time OBD-II

Live sensor polling at 2-second intervals. RPM, coolant temp, fuel trims, MAF, O2 sensors, timing advance — all streaming in real time.

Vehicle Health Score

Composite 0-100 score across 5 subsystems: engine, fuel system, emissions, cooling, and ignition. Instant letter grade for customers.

Root Cause Correlation

18+ multi-DTC pattern rules identify when separate codes share a single root cause. Goes beyond code-by-code analysis.

Dual-Screen Kiosk

Technician dashboard on a 720x720 HyperPixel touchscreen. Customer-facing 720x1280 portrait display. Both screens purpose-built.

VIN Decode

Reads VIN from the vehicle's ECU via OBD-II Mode 09, decodes year/make/model/engine through the NHTSA API. Automatic vehicle identification.

Stripe Payments

In-app repair estimates with Stripe Checkout integration. Customer pays on-screen. Webhook verification for reliability.

Diagnostic Reports

Print-ready reports with health breakdown, DTC details, root cause analysis, technical diagnosis, and sensor snapshots. Professional quality.

Service History

SQLite persistence tracks every session by VIN. Sensor logs, diagnosis records, and payment history. Full audit trail.

Workflow

Scan. Diagnose. Get paid.

A complete diagnostic session in four steps.

01

Connect

Plug the OBD-II adapter into the vehicle's diagnostic port. CWOP-Diag auto-detects the connection, reads the VIN, and starts polling live sensor data.

02

Scan & Analyze

Trouble codes are read and cross-referenced against 333+ DTC definitions. Root cause correlation identifies shared failures. Health score is computed across all subsystems.

03

AI Diagnosis

The local LLM receives a precisely structured context window and generates a technical analysis with specific diagnostic steps. A customer-friendly summary is generated for the second screen.

04

Estimate & Pay

The technician enters parts and labor. The estimate pushes to the customer screen. Customer pays via Stripe. Session is saved to the vehicle's service history.

Veepeak OBDCheck BLE+
Vehicle Interface

Bluetooth Low Energy OBD-II adapter. Plugs into any 1996+ vehicle's diagnostic port. No wiring required.
